# GPJ
GoPro Joiner - Joins chunked GoPro clips into one mp4-file for easier editing. The code is written in LiveScript.

## Installation

### Prerequisites

 * node.js 8.2.1 (as far as I know)
 * ffmpeg

### Install for CLI

    npm install -g gpj

### Install for node project

    npm install --save gpj

## Usage

### CLI

    gpj -h

### Code

    gpj = require("gpj")
